- C++ 是一門強大的編譯型語言，支援物件導向、泛型和底層記憶體操作，廣泛用於系統開發、遊戲引擎和高效能運算，學習曲線陡峭
- PHP 是一門專為 Web 開發設計的腳本語言，語法簡單易上手，擁有 Laravel 等成熟框架，主要用於伺服端網頁渲染和 CMS 開發，入門門檻低
- Node.js 是基於 V8 引擎的 JavaScript 執行環境，採用事件驅動和非阻塞 I/O 模型，適合建構高並發的網路應用和 API 服務，前端開發者可快速上手
- TypeScript 是 JavaScript 的超集，引入了靜態型別系統和介面等特性，提升了大型專案的可維護性，廣泛用於前端和 Node.js 後端開發，需要一定的型別系統理解
- Java 是一門跨平台的物件導向語言，擁有龐大的生態系和企業級框架，主要用於大型後端系統、Android 開發和微服務架構，語法嚴謹但學習資源豐富
- Rust 是一門注重記憶體安全的系統級語言，透過所有權機制實現零成本抽象，適用於系統程式設計、嵌入式和 WebAssembly，學習難度較高但安全性出色
- Go 是 Google 開發的靜態型別語言，以簡潔語法和內建並發支援著稱，主要用於雲端原生服務、微服務和 DevOps 工具開發，上手較快
- C# 是微軟主導的物件導向語言，與 .NET 生態系深度繫結，適用於 Windows 應用、遊戲開發（Unity）和企業級後端，語法類似 Java 但更現代
- Python 是一門動態型別的直譯語言，語法簡潔清晰，廣泛用於資料科學、人工智慧、自動化腳本和 Web 後端，入門簡單但精通需要領域知識


## 前端語言

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS-1572B6?style=flat&logo=css&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=white)

- HTML5 是最新的 HTML 標準，支援語意化標籤、音訊影片播放和 Canvas 繪圖，是所有網頁的結構基礎，入門簡單但需要熟悉標籤語意
- CSS 是用於描述網頁樣式的樣式表語言，支援 Flexbox、Grid 佈局和動畫效果，負責頁面的視覺呈現，基礎易學但深入佈局和響應式設計需要實作練習
- JavaScript 是 Web 的唯一原生腳本語言，支援事件處理、DOM 操作和非同步程式設計，用於實現網頁互動邏輯，語法靈活但非同步和原型鏈等概念需要理解


## 軟體和工具

![Git](https://img.shields.io/badge/Git-F05032?style=flat&logo=git&logoColor=white)
![Nginx](https://img.shields.io/badge/Nginx-009639?style=flat&logo=nginx&logoColor=white)
![MySQL](https://img.shields.io/badge/MySQL-4479A1?style=flat&logo=mysql&logoColor=white)
![Redis](https://img.shields.io/badge/Redis-DC382D?style=flat&logo=redis&logoColor=white)
![SQLite](https://img.shields.io/badge/SQLite-003B57?style=flat&logo=sqlite&logoColor=white)
![Docker](https://img.shields.io/badge/Docker-2496ED?style=flat&logo=docker&logoColor=white)
![GitHub Actions](https://img.shields.io/badge/GitHub%20Actions-2088FF?style=flat&logo=githubactions&logoColor=white)

- Git 是分散式版本控制系統，支援分支管理和離線提交，是程式碼協作和版本管理的事實標準，基礎操作簡單但進階用法（rebase、bisect）需要累積經驗
- Nginx 是高效能的 HTTP 伺服器和反向代理，以事件驅動架構實現低記憶體高並發，主要用於靜態資源託管、負載平衡和反向代理，設定語法需要熟悉
- MySQL 是最流行的開源關聯式資料庫，支援交易、索引和主從複製，廣泛用於 Web 應用的資料儲存，基礎 CRUD 簡單但效能調校需要經驗
- Redis 是基於記憶體的鍵值資料庫，支援字串、雜湊、列表等資料結構和發布訂閱，主要用於快取、工作階段管理和訊息佇列，指令簡單但架構設計需要理解
- SQLite 是輕量級的嵌入式關聯資料庫，無需獨立伺服器程序，資料儲存為單一檔案，適用於行動端、嵌入式和小型應用，使用極其簡單
- Docker 是容器化平台，透過映像檔和容器實現應用的打包與隔離部署，主要用於環境一致性保障和微服務部署，基礎指令易學但編排需要實作練習
- GitHub Actions 是 GitHub 內建的 CI/CD 平台，透過 YAML 工作流程實現自動化建置、測試和部署，與 GitHub 儲存庫深度整合，設定語法直觀易上手


## 開發框架

![Spring Boot](https://img.shields.io/badge/Spring%20Boot-6DB33F?style=flat&logo=springboot&logoColor=white)
![Vue.js](https://img.shields.io/badge/Vue.js-4FC08D?style=flat&logo=vuedotjs&logoColor=white)
![React](https://img.shields.io/badge/React-61DAFB?style=flat&logo=react&logoColor=white)

- Spring Boot 是基於 Spring 的快速開發框架，透過自動設定和約定優於設定簡化企業級應用開發，主要用於 Java 後端微服務和 RESTful API，需要一定的 Spring 基礎
- Vue.js 是漸進式前端框架，採用響應式資料繫結和元件化設計，易於與現有專案整合，廣泛用於單頁應用和中小型專案，上手友善且文件完善
- React 是 Facebook 開發的前端 UI 元件庫，基於 JSX 和虛擬 DOM 實現高效能渲染，擁有龐大的生態系（Next.js、React Native），適合大型應用但需要理解函數式程式設計思維
