- C++ はオブジェクト指向・ジェネリック・低レベルメモリ操作をサポートする強力なコンパイル型言語で、システム開発やゲームエンジン、高性能コンピューティングに広く使われています。学習曲線は急峻です
- PHP はWeb開発向けに設計されたスクリプト言語で、構文がシンプルで習得しやすく、Laravel などの成熟したフレームワークがあります。主にサーバーサイドのページレンダリングやCMS開発に使用され、参入障壁は低いです
- Node.js はV8エンジンベースのJavaScriptランタイムで、イベント駆動型のノンブロッキングI/Oモデルを採用しています。高同時接続のネットワークアプリやAPIサービスの構築に適しており、フロントエンド開発者でもすぐに使い始められます
- TypeScript はJavaScriptのスーパーセットで、静的型システムやインターフェースなどの機能を導入し、大規模プロジェクトの保守性を向上させます。フロントエンドやNode.jsのバックエンド開発に広く使われ、型システムの理解が必要です
- Java はクロスプラットフォームのオブジェクト指向言語で、膨大なエコシステムとエンタープライズ向けフレームワークを備えています。大規模バックエンドシステムやAndroid開発、マイクロサービスアーキテクチャに主に使用され、構文は厳格ですが学習リソースが豊富です
- Rust はメモリ安全性を重視するシステムレベル言語で、所有権メカニズムによりゼロコスト抽象化を実現します。システムプログラミングや組み込み、WebAssemblyに適しており、学習難易度は高いですが安全性に優れています
- Go はGoogleが開発した静的型言語で、簡潔な構文と組み込みの並行処理サポートが特徴です。クラウドネイティブサービスやマイクロサービス、DevOpsツール開発に主に使用され、習得は比較的容易です
- C# はマイクロソフト主導のオブジェクト指向言語で、.NETエコシステムと深く統合されています。Windowsアプリやゲーム開発（Unity）、エンタープライズ向けバックエンドに適しており、Javaに似た構文ですがよりモダンです
- Python は動的型付けのインタープリタ言語で、構文が簡潔で明快です。データサイエンスや人工知能、自動化スクリプト、Webバックエンドに広く使われ、入門は簡単ですが習熟には専門知識が必要です


## フロントエンド言語

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS-1572B6?style=flat&logo=css&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=white)

- HTML5 は最新のHTML標準で、セマンティックタグや音声・ビデオ再生、Canvas描画をサポートしています。すべてのWebページの構造基盤であり、入門は簡単ですがタグのセマンティクスに慣れる必要があります
- CSS はWebページのスタイルを記述するスタイルシート言語で、FlexboxやGridレイアウト、アニメーション効果をサポートしています。ページのビジュアル表現を担当し、基礎は習得しやすいですが、レイアウトやレスポンシブデザインの習熟には実践が必要です
- JavaScript はWebの唯一のネイティブスクリプト言語で、イベント処理やDOM操作、非同期プログラミングをサポートしています。Webページのインタラクションロジックの実装に使用され、構文は柔軟ですが非同期やプロトタイプチェーンなどの概念の理解が必要です


## ソフトウェアとツール

![Git](https://img.shields.io/badge/Git-F05032?style=flat&logo=git&logoColor=white)
![Nginx](https://img.shields.io/badge/Nginx-009639?style=flat&logo=nginx&logoColor=white)
![MySQL](https://img.shields.io/badge/MySQL-4479A1?style=flat&logo=mysql&logoColor=white)
![Redis](https://img.shields.io/badge/Redis-DC382D?style=flat&logo=redis&logoColor=white)
![SQLite](https://img.shields.io/badge/SQLite-003B57?style=flat&logo=sqlite&logoColor=white)
![Docker](https://img.shields.io/badge/Docker-2496ED?style=flat&logo=docker&logoColor=white)
![GitHub Actions](https://img.shields.io/badge/GitHub%20Actions-2088FF?style=flat&logo=githubactions&logoColor=white)

- Git は分散型バージョン管理システムで、ブランチ管理やオフラインコミットをサポートし、コードコラボレーションとバージョン管理の事実上の標準です。基本操作は簡単ですが、高度な使い方（rebase、bisect）には経験が必要です
- Nginx は高性能なHTTPサーバーおよびリバースプロキシで、イベント駆動アーキテクチャにより低メモリ・高同時接続を実現します。主に静的リソースのホスティングや負荷分散、リバースプロキシに使用され、設定構文に慣れる必要があります
- MySQL は最も人気のあるオープンソースリレーショナルデータベースで、トランザクションやインデックス、レプリケーションをサポートしています。Webアプリケーションのデータストレージに広く使われ、基本的なCRUDは簡単ですがパフォーマンスチューニングには経験が必要です
- Redis はメモリベースのキーバリューデータベースで、文字列やハッシュ、リストなどのデータ構造とPub/Subをサポートしています。主にキャッシュやセッション管理、メッセージキューに使用され、コマンドはシンプルですがアーキテクチャ設計の理解が必要です
- SQLite は軽量な組み込み型リレーショナルデータベースで、独立したサーバープロセスは不要で、データは単一ファイルに保存されます。モバイルや組み込み、小規模アプリケーションに適しており、使い方は非常に簡単です
- Docker はコンテナ化プラットフォームで、イメージとコンテナによりアプリケーションのパッケージ化と隔離デプロイを実現します。主に環境の一貫性確保とマイクロサービスのデプロイに使用され、基本コマンドは習得しやすいですがオーケストレーションには実践が必要です
- GitHub Actions はGitHub組み込みのCI/CDプラットフォームで、YAMLワークフローにより自動ビルド・テスト・デプロイを実現します。GitHubリポジトリと深く統合されており、設定構文は直感的で習得しやすいです


## 開発フレームワーク

![Spring Boot](https://img.shields.io/badge/Spring%20Boot-6DB33F?style=flat&logo=springboot&logoColor=white)
![Vue.js](https://img.shields.io/badge/Vue.js-4FC08D?style=flat&logo=vuedotjs&logoColor=white)
![React](https://img.shields.io/badge/React-61DAFB?style=flat&logo=react&logoColor=white)

- Spring Boot はSpringベースの高速開発フレームワークで、自動設定と規約による設定の簡素化によりエンタープライズ向けアプリ開発を効率化します。主にJavaバックエンドのマイクロサービスやRESTful APIに使用され、Springの基礎知識が必要です
- Vue.js はプログレッシブフロントエンドフレームワークで、リアクティブなデータバインディングとコンポーネントベースの設計を採用しています。既存プロジェクトへの統合が容易で、シングルページアプリケーションや中小規模プロジェクトに広く使われ、習得しやすくドキュメントも充実しています
- React はFacebookが開発したフロントエンドUIライブラリで、JSXと仮想DOMに基づく高効率レンダリングを実現しています。膨大なエコシステム（Next.js、React Native）を持ち、大規模アプリケーションに適していますが、関数型プログラミングの理解が必要です
