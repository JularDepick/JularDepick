- C++ — мощный компилируемый язык, поддерживающий объектно-ориентированное, обобщённое и низкоуровневое управление памятью, широко применяемый в системной разработке, игровых движках и высокопроизводительных вычислениях, с крутой кривой обучения
- PHP — язык сценариев, созданный специально для веб-разработки, с простым и доступным синтаксисом и зрелыми фреймворками, такими как Laravel, в основном используется для серверной отрисовки страниц и разработки CMS, с низким порогом входа
- Node.js — среда выполнения JavaScript на базе движка V8, использующая событийно-ориентированную модель с неблокирующим вводом-выводом, идеально подходит для создания высокопараллельных сетевых приложений и API-сервисов, легко осваивается фронтенд-разработчиками
- TypeScript — надмножество JavaScript, вводящее статическую систему типов и такие возможности, как интерфейсы, повышая удобство сопровождения крупных проектов, широко используется во фронтенд- и Node.js-бэкенд-разработке, требует некоторого понимания систем типов
- Java — кроссплатформенный объектно-ориентированный язык с огромной экосистемой и корпоративными фреймворками, в основном используемый для крупных бэкенд-систем, разработки под Android и микросервисных архитектур, синтаксис строгий, но учебных ресурсов много
- Rust — язык системного уровня с акцентом на безопасность памяти, реализующий абстракции с нулевой стоимостью через механизм владения, подходит для системного программирования, встроенных систем и WebAssembly, сложность обучения высокая, но безопасность превосходная
- Go — статически типизированный язык, разработанный Google, известный лаконичным синтаксисом и встроенной поддержкой параллелизма, в основном используется для облачных сервисов, микросервисов и DevOps-инструментов, осваивается быстро
- C# — объектно-ориентированный язык под руководством Microsoft, глубоко интегрированный с экосистемой .NET, подходит для Windows-приложений, разработки игр (Unity) и корпоративных бэкендов, синтаксис похож на Java, но более современный
- Python — динамически типизируемый интерпретируемый язык с лаконичным и понятным синтаксисом, широко используемый в науке о данных, искусственном интеллекте, автоматизации и веб-бэкендах, начать легко, но для мастерства нужны знания в предметной области


## Frontend-языки

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS-1572B6?style=flat&logo=css&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=white)

- HTML5 — последний стандарт HTML, поддерживающий семантические теги, воспроизведение аудио/видео и рисование на Canvas, является структурной основой всех веб-страниц, начать легко, но нужно разбираться в семантике тегов
- CSS — язык стилей для описания визуального оформления веб-страниц, поддерживающий Flexbox, Grid-раскладки и анимации, отвечает за внешний вид страниц, основы освоить легко, но освоение раскладок и адаптивного дизайна требует практики
- JavaScript — единственный собственный язык сценариев веба, поддерживающий обработку событий, манипуляции DOM и асинхронное программирование, используется для реализации интерактивной логики веб-страниц, синтаксис гибкий, но такие концепции, как асинхронность и прототипное наследование, требуют понимания


## Программное обеспечение и инструменты

![Git](https://img.shields.io/badge/Git-F05032?style=flat&logo=git&logoColor=white)
![Nginx](https://img.shields.io/badge/Nginx-009639?style=flat&logo=nginx&logoColor=white)
![MySQL](https://img.shields.io/badge/MySQL-4479A1?style=flat&logo=mysql&logoColor=white)
![Redis](https://img.shields.io/badge/Redis-DC382D?style=flat&logo=redis&logoColor=white)
![SQLite](https://img.shields.io/badge/SQLite-003B57?style=flat&logo=sqlite&logoColor=white)
![Docker](https://img.shields.io/badge/Docker-2496ED?style=flat&logo=docker&logoColor=white)
![GitHub Actions](https://img.shields.io/badge/GitHub%20Actions-2088FF?style=flat&logo=githubactions&logoColor=white)

- Git — распределённая система контроля версий, поддерживающая управление ветками и офлайн-коммиты, является фактическим стандартом для совместной работы с кодом и управления версиями, базовые операции просты, но продвинутые возможности (rebase, bisect) требуют накопления опыта
- Nginx — высокопроизводительный HTTP-сервер и обратный прокси, обеспечивающий низкое потребление памяти и высокую параллельность благодаря событийно-ориентированной архитектуре, используется для хостинга статических ресурсов, балансировки нагрузки и обратного проксирования, нужно ознакомиться с синтаксисом конфигурации
- MySQL — самая популярная открытая реляционная база данных, поддерживающая транзакции, индексы и репликацию «мастер-слейв», широко используется для хранения данных в веб-приложениях, базовый CRUD прост, но настройка производительности требует опыта
- Redis — база данных «ключ-значение» в памяти, поддерживающая такие структуры данных, как строки, хэши и списки, а также публикацию/подписку, используется для кэширования, управления сессиями и очередей сообщений, команды просты, но проектирование архитектуры требует понимания
- SQLite — лёгкая встраиваемая реляционная база данных, не требующая отдельного серверного процесса, данные хранятся в одном файле, подходит для мобильных, встроенных и небольших приложений, крайне проста в использовании
- Docker — платформа контейнеризации, упаковывающая и изолирующая приложения с помощью образов и контейнеров, используется для обеспечения единообразия среды и развёртывания микросервисов, базовые команды легко освоить, но оркестрация требует практики
- GitHub Actions — встроенная в GitHub платформа CI/CD, автоматизирующая сборку, тестирование и развёртывание через YAML-воркфлоу, глубоко интегрирована с GitHub-репозиториями, синтаксис конфигурации интуитивно понятен и прост для освоения


## Фреймворки разработки

![Spring Boot](https://img.shields.io/badge/Spring%20Boot-6DB33F?style=flat&logo=springboot&logoColor=white)
![Vue.js](https://img.shields.io/badge/Vue.js-4FC08D?style=flat&logo=vuedotjs&logoColor=white)
![React](https://img.shields.io/badge/React-61DAFB?style=flat&logo=react&logoColor=white)

- Spring Boot — фреймворк быстрой разработки на основе Spring, упрощающий создание корпоративных приложений благодаря автоматической конфигурации и принципу «согласование важнее конфигурации», используется для Java-бэкенд микросервисов и RESTful API, требует некоторой базы по Spring
- Vue.js — прогрессивный фронтенд-фреймворк, использующий реактивную привязку данных и компонентный дизайн, легко интегрируется с существующими проектами, широко используется для одностраничных приложений и небольших/средних проектов, удобен для начинающих и обладает отличной документацией
- React — фронтенд-библиотека интерфейсов, разработанная Facebook, обеспечивающая эффективный рендеринг через JSX и виртуальный DOM, с огромной экосистемой (Next.js, React Native), подходит для крупных приложений, но требует понимания концепций функционального программирования


> Приведённая информация основана исключительно на технологическом стеке, с которым я лично работал
