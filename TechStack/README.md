## 后端语言

![C++](https://img.shields.io/badge/C%2B%2B-00599C?style=flat&logo=cplusplus&logoColor=white)
![PHP](https://img.shields.io/badge/PHP-777BB4?style=flat&logo=php&logoColor=white)

![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat&logo=typescript&logoColor=white)

![Rust](https://img.shields.io/badge/Rust-333?style=flat&logo=rust&logoColor=white)

![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)

- C++ 是一门强大的编译型语言，支持面向对象、泛型和底层内存操作，广泛用于系统开发、游戏引擎和高性能计算，学习曲线陡峭
- PHP 是一门专为 Web 开发设计的脚本语言，语法简单易上手，拥有 Laravel 等成熟框架，主要用于服务端网页渲染和 CMS 开发，入门门槛低
- Node.js 是基于 V8 引擎的 JavaScript 运行时，采用事件驱动和非阻塞 I/O 模型，适合构建高并发的网络应用和 API 服务，前端开发者可快速上手
- TypeScript 是 JavaScript 的超集，引入了静态类型系统和接口等特性，提升了大型项目的可维护性，广泛用于前端和 Node.js 后端开发，需要一定的类型系统理解
- Java 是一门跨平台的面向对象语言，拥有庞大的生态和企业级框架，主要用于大型后端系统、安卓开发和微服务架构，语法严谨但学习资源丰富
- Rust 是一门注重内存安全的系统级语言，通过所有权机制实现零成本抽象，适用于系统编程、嵌入式和 WebAssembly，学习难度较高但安全性出色
- Go 是 Google 开发的静态类型语言，以简洁语法和内置并发支持著称，主要用于云原生服务、微服务和 DevOps 工具开发，上手较快
- C# 是微软主导的面向对象语言，与 .NET 生态深度绑定，适用于 Windows 应用、游戏开发（Unity）和企业级后端，语法类似 Java 但更现代
- Python 是一门动态类型的解释型语言，语法简洁清晰，广泛用于数据科学、人工智能、自动化脚本和 Web 后端，入门简单但精通需要领域知识


## 前端语言

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS-1572B6?style=flat&logo=css&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=white)

- HTML5 是最新的 HTML 标准，支持语义化标签、音视频播放和 Canvas 绘图，是所有网页的结构基础，入门简单但需要熟悉标签语义
- CSS 是用于描述网页样式的样式表语言，支持 Flexbox、Grid 布局和动画效果，负责页面的视觉呈现，基础易学但深入布局和响应式设计需要实践
- JavaScript 是 Web 的唯一原生脚本语言，支持事件处理、DOM 操作和异步编程，用于实现网页交互逻辑，语法灵活但异步和原型链等概念需要理解


## 软件和工具

![Git](https://img.shields.io/badge/Git-F05032?style=flat&logo=git&logoColor=white)
![Nginx](https://img.shields.io/badge/Nginx-009639?style=flat&logo=nginx&logoColor=white)
![MySQL](https://img.shields.io/badge/MySQL-4479A1?style=flat&logo=mysql&logoColor=white)
![Redis](https://img.shields.io/badge/Redis-DC382D?style=flat&logo=redis&logoColor=white)
![SQLite](https://img.shields.io/badge/SQLite-003B57?style=flat&logo=sqlite&logoColor=white)
![Docker](https://img.shields.io/badge/Docker-2496ED?style=flat&logo=docker&logoColor=white)
![GitHub Actions](https://img.shields.io/badge/GitHub%20Actions-2088FF?style=flat&logo=githubactions&logoColor=white)

- Git 是分布式版本控制系统，支持分支管理和离线提交，是代码协作和版本管理的事实标准，基础操作简单但高级用法（rebase、bisect）需要积累
- Nginx 是高性能的 HTTP 服务器和反向代理，以事件驱动架构实现低内存高并发，主要用于静态资源托管、负载均衡和反向代理，配置语法需要熟悉
- MySQL 是最流行的开源关系型数据库，支持事务、索引和主从复制，广泛用于 Web 应用的数据存储，基础 CRUD 简单但性能调优需要经验
- Redis 是基于内存的键值数据库，支持字符串、哈希、列表等数据结构和发布订阅，主要用于缓存、会话管理和消息队列，命令简单但架构设计需要理解
- SQLite 是轻量级的嵌入式关系数据库，无需独立服务器进程，数据存储为单个文件，适用于移动端、嵌入式和小型应用，使用极其简单
- Docker 是容器化平台，通过镜像和容器实现应用的打包与隔离部署，主要用于环境一致性保障和微服务部署，基础命令易学但编排需要实践
- GitHub Actions 是 GitHub 内置的 CI/CD 平台，通过 YAML 工作流实现自动化构建、测试和部署，与 GitHub 仓库深度集成，配置语法直观易上手


## 开发框架

![Spring Boot](https://img.shields.io/badge/Spring%20Boot-6DB33F?style=flat&logo=springboot&logoColor=white)
![Vue.js](https://img.shields.io/badge/Vue.js-4FC08D?style=flat&logo=vuedotjs&logoColor=white)
![React](https://img.shields.io/badge/React-61DAFB?style=flat&logo=react&logoColor=white)

- Spring Boot 是基于 Spring 的快速开发框架，通过自动配置和约定优于配置简化企业级应用开发，主要用于 Java 后端微服务和 RESTful API，需要一定的 Spring 基础
- Vue.js 是渐进式前端框架，采用响应式数据绑定和组件化设计，易于与现有项目集成，广泛用于单页应用和中小型项目，上手友好且文档完善
- React 是 Facebook 开发的前端 UI 库，基于 JSX 和虚拟 DOM 实现高效渲染，拥有庞大的生态（Next.js、React Native），适合大型应用但需要理解函数式编程思想
