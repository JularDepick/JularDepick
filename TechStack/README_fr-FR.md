![C++](https://img.shields.io/badge/C%2B%2B-00599C?style=flat&logo=cplusplus&logoColor=white)
![PHP](https://img.shields.io/badge/PHP-777BB4?style=flat&logo=php&logoColor=white)
![Node.js](https://img.shields.io/badge/Node.js-339933?style=flat&logo=nodedotjs&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat&logo=typescript&logoColor=white)
![Java](https://img.shields.io/badge/Java-ED8B00?style=flat&logo=openjdk&logoColor=white)
![Rust](https://img.shields.io/badge/Rust-333?style=flat&logo=rust&logoColor=white)

![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)

- C++ est un puissant langage compilé prenant en charge la programmation orientée objet, le générique et la manipulation bas niveau de la mémoire, largement utilisé pour le développement système, les moteurs de jeu et le calcul haute performance, avec une courbe d'apprentissage abrupte
- PHP est un langage de script conçu pour le développement Web, doté d'une syntaxe simple et accessible, avec des frameworks éprouvés comme Laravel, principalement utilisé pour le rendu côté serveur et le développement de CMS, avec une barrière d'entrée faible
- Node.js est un environnement d'exécution JavaScript basé sur le moteur V8, utilisant un modèle événementiel et des E/S non bloquantes, idéal pour construire des applications réseau à forte concurrence et des services API, rapidement maîtrisable par les développeurs front-end
- TypeScript est un sur-ensemble de JavaScript qui introduit un système de types statiques et des interfaces, améliorant la maintenabilité des grands projets, largement utilisé pour le développement front-end et back-end Node.js, nécessitant une certaine compréhension du système de types
- Java est un langage orientée objet et multiplateforme, disposant d'un écosystème vaste et de frameworks d'entreprise, principalement utilisé pour les grands systèmes back-end, le développement Android et les architectures de microservices, avec une syntaxe rigoureuse mais de nombreuses ressources d'apprentissage
- Rust est un langage système axé sur la sécurité mémoire, réalisant une abstraction sans coût grâce au mécanisme de propriété, adapté à la programmation système, l'embarqué et WebAssembly, avec une difficulté d'apprentissage élevée mais une sécurité remarquable
- Go est un langage à types statiques développé par Google, reconnu pour sa syntaxe concise et son support intégré de la concurrence, principalement utilisé pour les services cloud-natifs, les microservices et les outils DevOps, relativement facile à prendre en main
- C# est un langage orientée objet développé par Microsoft, étroitement lié à l'écosystème .NET, adapté aux applications Windows, au développement de jeux (Unity) et aux backends d'entreprise, avec une syntaxe similaire à Java mais plus moderne
- Python est un langage interprété à typage dynamique, doté d'une syntaxe concise et claire, largement utilisé en science des données, intelligence artificielle, scripts d'automatisation et back-end Web, facile à aborder mais nécessitant des connaissances spécifiques pour la maîtrise


## Langages front-end

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS-1572B6?style=flat&logo=css&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=white)

- HTML5 est la dernière norme HTML, prenant en charge les balises sémantiques, la lecture audio/vidéo et le dessin Canvas, constituant la base structurelle de toutes les pages Web, facile à aborder mais nécessitant une bonne connaissance de la sémantique des balises
- CSS est un langage de feuilles de style servant à décrire la présentation des pages Web, prenant en charge les mises en page Flexbox et Grid ainsi que les animations, responsable de l'apparence visuelle, simple à apprendre en base mais les mises en page avancées et le design adaptatif demandent de la pratique
- JavaScript est le seul langage de script natif du Web, prenant en charge la gestion des événements, la manipulation du DOM et la programmation asynchrone, utilisé pour implémenter la logique d'interaction des pages Web, avec une syntaxe flexible mais des concepts comme l'asynchrone et les prototypes nécessitant de la compréhension


## Logiciels et outils

![Git](https://img.shields.io/badge/Git-F05032?style=flat&logo=git&logoColor=white)
![Nginx](https://img.shields.io/badge/Nginx-009639?style=flat&logo=nginx&logoColor=white)
![MySQL](https://img.shields.io/badge/MySQL-4479A1?style=flat&logo=mysql&logoColor=white)
![Redis](https://img.shields.io/badge/Redis-DC382D?style=flat&logo=redis&logoColor=white)
![SQLite](https://img.shields.io/badge/SQLite-003B57?style=flat&logo=sqlite&logoColor=white)
![Docker](https://img.shields.io/badge/Docker-2496ED?style=flat&logo=docker&logoColor=white)
![GitHub Actions](https://img.shields.io/badge/GitHub%20Actions-2088FF?style=flat&logo=githubactions&logoColor=white)

- Git est un système de contrôle de version distribué, prenant en charge la gestion de branches et les commits hors ligne, standard de facto pour la collaboration et la gestion de versions du code, avec des opérations de base simples mais des usages avancés (rebase, bisect) nécessitant de l'expérience
- Nginx est un serveur HTTP et proxy inverse haute performance, utilisant une architecture événementielle pour une faible consommation mémoire et une forte concurrence, principalement utilisé pour l'hébergement de ressources statiques, l'équilibrage de charge et le proxy inverse, avec une syntaxe de configuration à maîtriser
- MySQL est la base de données relationnelle open source la plus populaire, prenant en charge les transactions, les index et la réplication maître-esclave, largement utilisée pour le stockage de données des applications Web, avec des opérations CRUD de base simples mais l'optimisation des performances nécessitant de l'expérience
- Redis est une base de données clé-valeur en mémoire, prenant en charge les chaînes, les hachages, les listes et d'autres structures de données ainsi que la publication-abonnement, principalement utilisée pour la mise en cache, la gestion de sessions et les files d'attente de messages, avec des commandes simples mais une conception d'architecture nécessitant de la compréhension
- SQLite est une base de données relationnelle légère et embarquée, ne nécessitant pas de processus serveur distinct, stockant les données dans un seul fichier, adaptée aux appareils mobiles, aux systèmes embarqués et aux petites applications, extrêmement simple à utiliser
- Docker est une plateforme de conteneurisation, permettant l'empaquetage et le déploiement isolé des applications via des images et des conteneurs, principalement utilisée pour garantir la cohérence des environnements et le déploiement de microservices, avec des commandes de base faciles à apprendre mais l'orchestration nécessitant de la pratique
- GitHub Actions est la plateforme CI/CD intégrée à GitHub, permettant l'automatisation des builds, des tests et des déploiements via des workflows YAML, profondément intégrée aux dépôts GitHub, avec une syntaxe de configuration intuitive et accessible


## Frameworks de développement

![Spring Boot](https://img.shields.io/badge/Spring%20Boot-6DB33F?style=flat&logo=springboot&logoColor=white)
![Vue.js](https://img.shields.io/badge/Vue.js-4FC08D?style=flat&logo=vuedotjs&logoColor=white)
![React](https://img.shields.io/badge/React-61DAFB?style=flat&logo=react&logoColor=white)

- Spring Boot est un framework de développement rapide basé sur Spring, simplifiant le développement d'applications d'entreprise grâce à la configuration automatique et la convention plutôt que la configuration, principalement utilisé pour les microservices back-end Java et les API RESTful, nécessitant certaines bases sur Spring
- Vue.js est un framework front-end progressif, utilisant la liaison de données réactive et la conception par composants, facile à intégrer dans des projets existants, largement utilisé pour les applications monopage et les projets de petite à moyenne taille, avec une courbe d'apprentissage douce et une excellente documentation
- React est une bibliothèque d'interface utilisateur front-end développée par Facebook, basée sur le JSX et le DOM virtuel pour un rendu efficace, disposant d'un écosystème vaste (Next.js, React Native), adapté aux grandes applications mais nécessitant la compréhension de la programmation fonctionnelle
