## Backend Languages

![C++](https://img.shields.io/badge/C%2B%2B-00599C?style=flat&logo=cplusplus&logoColor=white)
![PHP](https://img.shields.io/badge/PHP-777BB4?style=flat&logo=php&logoColor=white)
![Node.js](https://img.shields.io/badge/Node.js-339933?style=flat&logo=nodedotjs&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat&logo=typescript&logoColor=white)
![Java](https://img.shields.io/badge/Java-ED8B00?style=flat&logo=openjdk&logoColor=white)
![Rust](https://img.shields.io/badge/Rust-333?style=flat&logo=rust&logoColor=white)
![Go](https://img.shields.io/badge/Go-00ADD8?style=flat&logo=go&logoColor=white)
![C#](https://img.shields.io/badge/C%23-239120?style=flat&logo=dotnet&logoColor=white)
![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)

- C++ is a powerful compiled language that supports object-oriented, generic, and low-level memory operations, widely used in system development, game engines, and high-performance computing, with a steep learning curve
- PHP is a scripting language designed specifically for web development, with simple and accessible syntax and mature frameworks like Laravel, primarily used for server-side page rendering and CMS development, with a low barrier to entry
- Node.js is a JavaScript runtime built on the V8 engine, using an event-driven and non-blocking I/O model, ideal for building highly concurrent network applications and API services, easy for frontend developers to pick up
- TypeScript is a superset of JavaScript that introduces a static type system and features like interfaces, improving maintainability for large-scale projects, widely used in frontend and Node.js backend development, requiring some understanding of type systems
- Java is a cross-platform, object-oriented language with a vast ecosystem and enterprise-grade frameworks, primarily used for large-scale backend systems, Android development, and microservice architectures, with rigorous syntax but abundant learning resources
- Rust is a system-level language focused on memory safety, achieving zero-cost abstractions through its ownership mechanism, suitable for systems programming, embedded systems, and WebAssembly, with a high learning difficulty but outstanding safety guarantees
- Go is a statically typed language developed by Google, known for its clean syntax and built-in concurrency support, primarily used for cloud-native services, microservices, and DevOps tooling, relatively quick to learn
- C# is an object-oriented language led by Microsoft, deeply integrated with the .NET ecosystem, suitable for Windows applications, game development (Unity), and enterprise-grade backends, with syntax similar to Java but more modern
- Python is a dynamically typed, interpreted language with clean and concise syntax, widely used in data science, artificial intelligence, automation scripting, and web backends, easy to get started but requires domain expertise to master


## Frontend Languages

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS-1572B6?style=flat&logo=css&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=white)

- HTML5 is the latest HTML standard, supporting semantic tags, audio/video playback, and Canvas drawing, serving as the structural foundation of all web pages, easy to get started but requires familiarity with tag semantics
- CSS is a stylesheet language used to describe the visual presentation of web pages, supporting Flexbox, Grid layouts, and animations, responsible for the visual appearance of pages, easy to learn the basics but mastering layouts and responsive design takes practice
- JavaScript is the only native scripting language of the web, supporting event handling, DOM manipulation, and asynchronous programming, used to implement interactive web logic, with flexible syntax but concepts like asynchronous programming and the prototype chain that require understanding


## Software and Tools

![Git](https://img.shields.io/badge/Git-F05032?style=flat&logo=git&logoColor=white)
![Nginx](https://img.shields.io/badge/Nginx-009639?style=flat&logo=nginx&logoColor=white)
![MySQL](https://img.shields.io/badge/MySQL-4479A1?style=flat&logo=mysql&logoColor=white)
![Redis](https://img.shields.io/badge/Redis-DC382D?style=flat&logo=redis&logoColor=white)
![SQLite](https://img.shields.io/badge/SQLite-003B57?style=flat&logo=sqlite&logoColor=white)
![Docker](https://img.shields.io/badge/Docker-2496ED?style=flat&logo=docker&logoColor=white)
![GitHub Actions](https://img.shields.io/badge/GitHub%20Actions-2088FF?style=flat&logo=githubactions&logoColor=white)

- Git is a distributed version control system that supports branch management and offline commits, serving as the de facto standard for code collaboration and version management, with simple basic operations but advanced usage (rebase, bisect) that takes time to master
- Nginx is a high-performance HTTP server and reverse proxy that achieves low memory usage and high concurrency through its event-driven architecture, primarily used for static resource hosting, load balancing, and reverse proxying, requiring familiarity with its configuration syntax
- MySQL is the most popular open-source relational database, supporting transactions, indexing, and primary-replica replication, widely used for data storage in web applications, with simple basic CRUD but performance tuning that requires experience
- Redis is an in-memory key-value database supporting data structures like strings, hashes, and lists as well as pub/sub messaging, primarily used for caching, session management, and message queues, with simple commands but architecture design that requires understanding
- SQLite is a lightweight embedded relational database that requires no standalone server process, storing data in a single file, suitable for mobile, embedded, and small-scale applications, extremely easy to use
- Docker is a containerization platform that packages and isolates applications through images and containers, primarily used for ensuring environment consistency and deploying microservices, with basic commands that are easy to learn but orchestration that requires practice
- GitHub Actions is GitHub's built-in CI/CD platform that automates builds, testing, and deployment through YAML workflows, deeply integrated with GitHub repositories, with an intuitive configuration syntax that is easy to pick up


## Development Frameworks

![Spring Boot](https://img.shields.io/badge/Spring%20Boot-6DB33F?style=flat&logo=springboot&logoColor=white)
![Vue.js](https://img.shields.io/badge/Vue.js-4FC08D?style=flat&logo=vuedotjs&logoColor=white)
![React](https://img.shields.io/badge/React-61DAFB?style=flat&logo=react&logoColor=white)

- Spring Boot is a rapid development framework built on Spring, simplifying enterprise application development through auto-configuration and convention over configuration, primarily used for Java backend microservices and RESTful APIs, requiring some Spring background
- Vue.js is a progressive frontend framework that uses reactive data binding and component-based design, easy to integrate with existing projects, widely used for single-page applications and small-to-medium projects, beginner-friendly with excellent documentation
- React is a frontend UI library developed by Facebook, achieving efficient rendering through JSX and a virtual DOM, with a vast ecosystem (Next.js, React Native), suitable for large-scale applications but requires an understanding of functional programming concepts
